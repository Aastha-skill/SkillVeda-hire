PART 6 — GOOGLE OAUTH
─────────────────────────────────────────

In server/hiring-routes.ts add:

GET /api/hiring/auth/google
- Build Google OAuth URL with these params:
  client_id: process.env.GOOGLE_CLIENT_ID
  redirect_uri: process.env.GOOGLE_REDIRECT_URI
  response_type: code
  scope: email profile
  access_type: offline
- Redirect to Google OAuth URL

GET /api/hiring/auth/google/callback
- Get code from query params
- Exchange code for tokens via:
  POST https://oauth2.googleapis.com/token
  with client_id, client_secret, 
  redirect_uri, code, grant_type
- Get user info from:
  GET https://www.googleapis.com/oauth2/v2/userinfo
  with access_token
- Check if company exists with this email
- If exists → update googleId, generate JWT
- If not exists → create new company:
  companyName: Google profile name + " (Google)"
  contactName: Google profile name
  email: Google email
  credits: 20
  googleId: Google sub/id
- Generate JWT token (30 days)
- Redirect to:
  /hire/dashboard?token=[JWT]&company=[JSON]

In client/src/pages/hire-login.tsx
Add this useEffect to handle Google callback:

useEffect(() => {

- Return all companies:
  id, companyName, contactName, email, 
  credits, plan, createdAt

POST /api/hiring/admin/credits
Header: x-admin-key: [value from env ADMIN_SECRET]

- Subtract: credits = max(0, credits - amount)
- Set: credits = amount
- Return updated company
