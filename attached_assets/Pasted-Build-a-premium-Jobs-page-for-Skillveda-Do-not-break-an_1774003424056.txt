STEP 1 — TWO NEW API ENDPOINTS
Add both to server/routes.ts
════════════════════════════════════

POST /api/candidate/register
Body: { fullName, email, phone }
- If email not in student_profiles: INSERT
- INSERT into leads: 
  { fullName, email, phone, 
    leadType:"JOB", sourcePage:"jobs-page" }
- Return jwt.sign({email, name:fullName}, 
  JWT_SECRET, {expiresIn:"30d"})
  as { token, candidate:{name,email,phone} }

GET /api/candidate/verify-token  
Header: Authorization: Bearer <token>
- Verify JWT → return {valid:true, candidate}
  or {valid:false}
